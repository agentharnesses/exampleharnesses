---
description: Skills for gathering and storing external information needed before working on an application — job descriptions, company research, and role context.
---

- **job-description/** — Fetch or accept a job description and save it to `references/job-descriptions/`. Use when the user wants to load a posting before tailoring materials or assessing fit.
- **find-listings/** — Search the web for new job postings matching the target profile and surface them as candidate leads. Use when the user wants to discover new roles rather than work an existing lead.
