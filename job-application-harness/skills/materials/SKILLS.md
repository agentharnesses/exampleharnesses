---
description: Skills for drafting and tailoring application materials such as resumes and cover letters.
---

- **docx/** — Create, edit, and analyze `.docx` files. Use when producing or tailoring resume documents in Word format.
- **tailor/** — Generate a tailored cover letter, core competencies list, and skills list for a specific job posting by deriving content from existing resumes. Use when the user provides a job listing and wants new application materials.
- **answer/** — Answer open-ended application questions by adapting existing answers from `references/answers/`. Use when the user pastes a free-text question from a job application form. Saves each answer for future reuse.
