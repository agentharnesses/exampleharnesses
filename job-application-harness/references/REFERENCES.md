---
description: Reference documents for the job application workflow.
---

- **leads.csv** — Application pipeline tracker: company, role, posting URL, status, applied date, resume used, compensation, location
- **job-descriptions/** — Saved job postings, one markdown file per role. Used as input for tailoring materials and assessing fit.
- **resumes/** — Tailored resumes, one per target role or application
- **target-roles.md** — Profile of target roles, preferred industries, company size, and criteria for evaluating fit
